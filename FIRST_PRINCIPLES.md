# First-Principles Research Charter

本项目的核心宗旨：

> 一切从第一性原理出发进行分析、建模、实验和解释。

## 第一性原理分解

我们不预设存在“思考神经元”。从最小事实出发：

1. 自回归语言模型在每一步根据当前隐藏状态产生下一 token 分布。
2. “继续推理”和“立即作答”首先表现为下一 token 分布的可测差异。
3. 如果某个内部方向具有因果作用，沿该方向干预隐藏状态应可重复地改变该分布。
4. “有用 effort”不能等同于更长输出；它必须同时满足：
   - 在需要推理的任务上提高或至少不降低正确性；
   - 增加继续推理倾向时不导致无控制 verbosity；
   - 正向和反向干预具有方向性；
   - 随机方向、verbosity 方向和任务外对照不能同样解释效果。

## 实验优先级

CPU 环境下完整自回归 sweep 很慢，因此先用更基础、更便宜的因果问题：

```text
高 effort prompt 与低 effort prompt 的隐藏状态差分方向，
是否能在 neutral prompt 上提高“继续推理 token”相对“立即作答 token”的下一步概率？
```

这个实验不是最终结论，而是第一层因果筛选：

- 若不成立：说明当前 direction 甚至不能改变局部 token 动力学，应返回重构 prompt、数据或层位。
- 若成立：再进入生成长度、正确率、flip、随机方向和闭环 CET 实验。

## 停止条件

除非出现以下任一情况，否则继续迭代实验：

1. 多轮实验仍无法找到可复现信号，并且失败原因被记录为证据；
2. 找到一个可复现、带对照的内部干预信号，并形成报告。
